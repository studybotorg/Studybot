# 📚 StudyBot AI – Pakistan

StudyBot AI is a sleek, AI-powered study companion designed specifically for students in Pakistan. It provides a localized educational experience with dual-language support, multi-subject focusing, and an immersive interface built to enhance the daily study routine.

---

## ✨ Key Features

- **🔑 Secure Authentication:** Powered by **Supabase Auth** (with fallback handling), offering a beautiful, blurred glassmorphic sign-in and sign-up card featuring an option for persistent login storage.
- **🎙️ Localized & Voice Ready:** Integrated Voice-to-Text (`.mic-btn`) featuring microphone pulse animations to accommodate verbal queries.
- **🌐 Dual-Language Support:** Instant toggling between **English** and **Urdu (نستعلیق - Noto Nastaliq Urdu)**, aligning the layout and text properties directionally (RTL) for perfect local script rendering.
- **📐 Mathematical Equations Rendering:** Out-of-the-box support for complex equations and scientific queries utilizing embedded **KaTeX JS/CSS**.
- **📚 Dynamic Subject Picker:** A multi-grid subject selection overlay allowing students to tailor AI behavior according to their current course of study.
- **⚙️ Immersive Settings Board:** In-app controls to manage API keys, toggle profiles, monitor study streaks, and manage application configurations seamlessly.
- **📱 Responsive Layout:** Optimized fully for mobile and desktop screens, including collapsible navigation sidebars and custom gesture-overlays.

---

## 🛠️ Tech Stack

- **Frontend:** Semantic HTML5, Custom CSS3 Variables (with Dark Mode presets), Vanilla JavaScript (ES6+).
- **Authentication & Backend:** [Supabase](https://supabase.com/) JavaScript Client (Modular Integration).
- **Typography:** Google Fonts (*Plus Jakarta Sans*, *Baloo 2*, *Noto Nastaliq Urdu*).
- **Libraries Engine:** KaTeX CDN (High-speed math rendering).
